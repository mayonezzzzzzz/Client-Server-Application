Список параметров: 
--------------------------------------------------------------------------------------------------------------------
     Название / сокращение |  Значение по умолчанию   | Описание
--------------------------------------------------------------------------------------------------------------------
       --help / --h        |                          | Выводит описание и значение по умолчанию для всех параметров
    --address / --a        | 127.0.0.1                | Адрес сервера
       --port / --p        | 8080                     | Порт для соединения с сервером
--images-path / --i        | C:\Users\Public\Pictures | Путь для поиска изображений для отправки
       --text / --t        | default text             | Текст для наложения на изображение
--output-path / --out      | C:\Users\Public\Pictures | Путь для сохранения полученных ответов от сервера
--------------------------------------------------------------------------------------------------------------------

Каждый из параметров задается пользователем при запуске приложения.
